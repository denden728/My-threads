🧠 このアカウントの核心マインド

💬
売るな、共感しろ

「買って！」じゃなく「これ良かったよ」。座り仕事の仲間にLINEで教える感覚が正解。営業マンになった瞬間、人は離れる。

📊
数字で語れ

「良い」「最高」は誰でも言える。「ウエスト-4cm」「間食0回」「体脂肪率-3%」が刺さる。体験を数値化せよ。

🔄
量→改善→質

最初から完璧は不可能。まず30投稿してデータを取る。データが勝ちパターンを教えてくれる。

🎯
座りっぱなしの人だけに届ける

「みなさん」ではなく「1日8時間座って腹が出てきた32歳のあなた」。たった1人に向けて書くから刺さる。

⏰
継続が最強の戦略

1ヶ月で辞める人が9割。3ヶ月続けるだけで上位10%に入る。才能じゃない、続けた者が勝つ。

⚖️
薬機法を守れ

「痩せます」「効果あります」は絶対NG。「私の場合は-3kgだった」「体感として楽になった」が正解。個人の体験として書く。

🔥 こんな時どうする？（シチュエーション別）

😰 投稿しても全然見られない▼
  • 🎣 フックが弱い → フック&CTAタブで「共感あるある型」を試す
  • ⏰ 投稿時間がズレてる → 20:00〜22:00に投稿
  • 🏷️ ハッシュタグが足りない → リサーチタブで生成
  • 👥 フォロワーとの交流が少ない → 1日5件コメント
  • 📝 投稿が短すぎる → 長文生成タブを活用
😭 見られるけどクリックされない▼
  • 🎯 CTAが弱い → 「同志型CTA」に変更してみる
  • 📊 ベネフィットが曖昧 → 売上特化モードで数字訴求
  • 🤝 信頼感がない → 自分の体験数字を入れる
  • 🔗 リンクの位置 → コメント欄に貼る方がクリック率UP
💀 シャドウバンされた気がする▼
  • ⏸️ 2〜3日投稿を休む
  • 🚫 NGチェックリストを確認（薬機法表現も）
  • 📉 アフィ投稿の比率を下げる
  • ⏰ 投稿間隔を最低3時間以上空ける
  • 🔄 復帰時は有益投稿から
🤔 ネタ切れ▼
  • 💡 リサーチタブでネタ30個生成（8ジャンルあり）
  • 🧬 バズ設計タブで他ジャンルのバズ投稿を横展開
  • 📅 季節カレンダーで今月の狙い目を確認
  • 📊 分析タブで勝ちパターンの切り口を変えて再投稿
😓 フォロワーが増えない▼
  • 📚 有益投稿の比率を上げる（デスクワーカーあるある系）
  • 🎣 バズ職人タブで拡散型投稿を量産
  • 📝 長文投稿で滞在時間を稼ぐ
  • 💬 他のデスクワーカー系投稿に価値あるコメント
💸 収益が出ない▼
  • 🎯 高単価グッズ（EMS、マッサージガン等）の比率を上げる
  • 💰 25,000円以上の商品で報酬上限1,000円を狙う
  • 📊 クリック率の高いパターンをグッズ系で量産
  • 🔥 楽天セール期間に集中投稿
  • 📈 料率アップ商品を週1でチェック

📈 収益アップの方程式

収益 = 表示回数 × クリック率 × 購入率 × 報酬単価
表示回数↑ → バズ職人 / 長文投稿 / 20:00〜22:00投稿
クリック率↑ → 共感フック / 数字訴求 / 体験ベース
購入率↑ → 高レビュー商品 / セール期間
報酬単価↑ → 25,000円以上の高単価グッズ / 料率アップ商品
