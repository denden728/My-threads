🧠 アフィリエイトの核心マインド

💬
売るな、教えろ

「買って！」じゃなく「これ良かったよ」。友達にLINEで教える感覚が正解。営業マンになった瞬間、人は離れる。

📊
数字で語れ

「良い」「最高」は誰でも言える。「入眠20分→7分」「電気代が月3,000円減」が刺さる。体験を数値化せよ。

🔄
量→改善→質

最初から完璧は不可能。まず30投稿してデータを取る。データが勝ちパターンを教えてくれる。

🎯
1人に届ける

「みなさん」ではなく「朝起きると首が痛い30代のあなた」。たった1人に向けて書くから刺さる。

⏰
継続が最強の戦略

1ヶ月で辞める人が9割。3ヶ月続けるだけで上位10%に入る。才能じゃない、続けた者が勝つ。

🚫
AI臭を消せ

「です・ます」「〜になります」「おすすめです」→全部NG。「まじでこれ良い」「使ってみて」が正解。

🔥 こんな時どうする？（シチュエーション別）

😰 投稿しても全然見られない▼
  • 🎣 フックが弱い → フック&CTAタブで強化
  • ⏰ 投稿時間がズレてる → 20:00〜22:00に投稿
  • 🏷️ ハッシュタグが足りない → リサーチタブで生成
  • 👥 フォロワーとの交流が少ない → 1日5件コメント
  • 📝 投稿が短すぎる → 長文生成タブを活用
😭 見られるけどクリックされない▼
  • 🎯 CTAが弱い → フック&CTAタブで変更
  • 📊 ベネフィットが曖昧 → 売上特化モードで数字訴求
  • 🤝 信頼感がない → 体験談を入れる
  • 🔗 リンクの位置 → コメント欄に貼る
💀 シャドウバンされた気がする▼
  • ⏸️ 2〜3日投稿を休む
  • 🚫 NGチェックリストを確認
  • 📉 アフィ投稿の比率を下げる
  • ⏰ 投稿間隔を最低3時間以上空ける
  • 🔄 復帰時は有益投稿から
🤔 ネタ切れ▼
  • 💡 リサーチタブでネタ30個生成
  • 🧬 バズ設計タブで横展開
  • 📅 季節カレンダーを活用
  • 📊 分析タブで勝ちパターン横展開
😓 フォロワーが増えない▼
  • 📚 有益投稿の比率を上げる
  • 🎣 バズ職人タブで拡散型投稿
  • 📝 長文投稿で滞在時間UP
  • 💬 価値あるコメントを残す
💸 収益が出ない▼
  • 🎯 報酬率8%以上の商品を選ぶ
  • 💰 単価5,000円以上に切り替え
  • 📊 クリック率の高いパターンを量産
  • 🔥 セール期間に集中投稿

📈 収益アップの方程式

収益 = 表示回数 × クリック率 × 購入率 × 報酬単価
表示回数↑ → バズ職人 / 長文投稿 / 最適時間帯
クリック率↑ → フック&CTA / 体験ベース / 数字訴求
購入率↑ → 高レビュー商品 / セール期間
報酬単価↑ → 高単価商品 / 報酬率8%以上

💎 忘れちゃいけない鉄則10

1投稿間隔は最低2〜3時間空ける
2PR表記は絶対忘れるな（#PR #広告）
3「です・ます」は使わない。友達口調で。
4同じ商品を連投しない（最低3投稿空ける）
5コメントには必ず返信する
6アフィ投稿は全体の半分以下に
7リンクはコメント欄に貼る方がクリック率UP
8具体的な数字を入れる（○日で○○になった）
9投稿前に「声に出して読んで」自然か確認
101日6投稿を30日続けることが最重要
